## Setup

\`\`\`bash
python3 -m venv lc-academy-env
source lc-academy-env/bin/activate
pip install -r requirements.txt
cp .env.example .env # add your API keys
\`\`\`

Requires Python 3.11, 3.12, or 3.13.
